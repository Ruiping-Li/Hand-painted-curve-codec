# Hand-painted-curve-codec
Coding and decoding of hand-painted curve in Matlab with GUI

## 代码说明
+ code文件夹中为输出好的可执行文件，其他为原工程文件
+ 非win7 64 bit或matlab版本过低会可能会有兼容问题，需要将代码复制进MATLAB中重新构建即可
